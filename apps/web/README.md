# Equity Builders (web)

This folder contains the Next.js (App Router) application for the Equity Builders platform.

## Run

From the repository root:

```bash
npm install
npm run dev
```

## Key routes

- `/`: landing
- `/login`: demo sign-in (disable with `EB_DEMO_MODE=0`)
- `/app`: authenticated shell
- `/app/intelligence`: Intelligence Center + ATOS panel
